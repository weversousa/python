# Operadores de associação

Os operadores de associação são utilizados para verificar se uma sequência contém um objeto.

# em

*Exemplo*:
~~~python
print('x' in 'Python')
print('y' in 'Python')
~~~

*Saída*:
~~~python
False
True
~~~

# não em

*Exemplo*:
~~~python
print('x' not in 'Python')
print('y' not in 'Python')
~~~

*Saída*:
~~~python
True
False
~~~
